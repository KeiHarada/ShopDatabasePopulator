---------------------------------------I. REQUIREMENTS---------------------------------------

To run this program JAVA is required, it was developed in JAVA 1.8.0_144, though earlier versions
may be good enough, this has not been tested.

The project conducts some HTTP requests to get shop information in Santander city. 
And it should be compiled and run along the MySQL driver.

Making it possible, following external jar-file is needed:
mysql-connector-java-5.1.40-bin
json-20160810.jar
